# InstaPDF 📄✨

**InstaPDF** is a premium, client-side web application designed to merge and convert multiple images (JPG, PNG, WebP) into a single, highly-optimized PDF document instantly. It is particularly designed for stitching document scans, ID cards, and CNICs (front/back) seamlessly.

---

## 🚀 Key Features

*   **🔒 100% Private & Client-Side:** Yourzz files never leave your computer. All image processing, rotation, compression, and PDF compilation are done directly in the browser via JavaScript.
*   **🔄 Drag & Drop Reordering:** Arrange your pages effortlessly by dragging and dropping thumbnails in the interactive workspace grid.
*   **↩️ Rotation Controls:** Rotate individual images by 90-degree increments directly inside the workspace before compiling to correct any vertical/horizontal alignment issues.
*   **⚙️ Advanced PDF Settings:**
    *   **Auto Page Size:** Automatically fits page dimensions to each individual image—ideal for neat, margin-less CNIC scans.
    *   **A4/Letter Presets:** Fit images proportionally into standard paper layouts.
    *   **Custom Margins:** Specify page padding/margins in millimeters (mm).
    *   **Orientation Toggle:** Switch between portrait and landscape modes.
    *   **DPI Compression Options:** Choose from *Original*, *High (Recommended)*, *Medium*, or *Low* quality to customize output file size and resolution.
*   **🎨 Premium Modern UI:** Built with sleek glassmorphism styling, fully responsive grids, interactive micro-animations, a theme toggle (Dark/Light mode), and a step-by-step progress/loading indicator.

---

## 🛠️ Tech Stack

*   **Core Logic:** Vanilla JavaScript (ES6+), HTML5 Canvas API
*   **Styling:** Modern CSS (CSS variables, Flexbox, Grid, Glassmorphic panels)
*   **Icons:** FontAwesome v6
*   **PDF Engine:** jsPDF (included via CDN)
